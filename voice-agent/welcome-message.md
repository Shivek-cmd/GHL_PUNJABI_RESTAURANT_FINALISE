# Welcome Message — Sonu da Dhaba Voice Agent

---

## Inbound Welcome Message

> Paste this exactly into GHL → Settings → Voice AI → Welcome Message (Inbound)

```
Hey, thanks for calling Sonu da Dhaba! Would you like to continue in English, Hindi, or Punjabi?
```

---

## Why This Works

- Short, natural, and to the point
- Explicitly asks for language so there's no guessing
- Agent locks into whichever language the caller picks and stays in it for the full call
